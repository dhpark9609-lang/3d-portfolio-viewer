## 3D Viewer Controls

- **Left Click + Drag**: Rotate model
- **Right Click + Drag**: Rotate model
- **Alt + Right Click + Drag**: Zoom in/out
- **Middle Click + Drag**: Pan camera
- **Mouse Wheel**: Zoom
- **F Key**: Frame focus on model
